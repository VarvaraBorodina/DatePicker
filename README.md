# Date Picker

### Instalation

```
npm i calendar-with-todo-list
```

### Components

```
<Calendar />
```

### Props

```
type?: 0 | 1 | 2 (0 - week, 1 - month (default), 2 - year)
color?: string
min?: Date
max?: Date
firstDayOfWeek?: 0 | 1 (0 - sunday, 1 - monday (default))
todoList?: boolean
weekends?: boolean
daysOff?: boolean
```
